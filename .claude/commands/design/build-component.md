# Build Component

Create a new React component following Compilo Design System.

$ARGUMENTS

## Process

Delegate to @design-orchestrator with this request:

1. @component-builder creates the component
2. @styling-agent verifies Tailwind compliance
3. @accessibility-auditor checks WCAG compliance
4. @design-reviewer validates final output

## Output

- Component file at packages/ui/src/components/
- Updated exports in packages/ui/src/index.ts
- Usage documentation
- Compliance checklist
