# Design System Review

Review code for Compilo Design System compliance.

## Target

$ARGUMENTS

## Process

Delegate to @design-reviewer agent with the specified path.

The reviewer will:

1. Scan for hard-coded colors
2. Scan for arbitrary spacing
3. Check accessibility issues
4. Verify component patterns
5. Verify if the design complies with the compilo design in terms of look and feel

## Output

Generate compliance report with:

- Summary table
- Critical issues with line numbers
- Fixes for each violation
- Recommendations
