# Claude Command: Lint

Run comprehensive linting and type checking, then fix all issues found.

## Usage

```
/lint [ file path ]
```

## Process

First: use the eslint mcp "$ARGUMENTS"' AND check for any typescript errors.
STEP 1: Output the results.
STEP 2: Deep analysis via thinkdeep ("ultrathink") of the issues.
STEP 3: Fix categorized issues with Read/Edit tools.
STEP 4: Re-run all linters to confirm.
